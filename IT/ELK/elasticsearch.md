# Elasticsearch

> elasticsearch 是面向文档关系行数据库,一切都是 JSON

## 概念

1. 关系型数据库与 elasticsearch 相关概念

| mysql            | elasticsearch        |
| ---------------- | -------------------- |
| 数据库 databases | 索引 indices         |
| 表 tables        | 类型 types(逐步弃用) |
| 行 rows          | 文档 documents       |
| 字段 columns     | fields               |
